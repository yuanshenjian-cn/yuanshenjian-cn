# User Input 目录

用于存放用户提供的素材文件，供 blog-writer 写作时参考。

## 使用规则

**只能在用户明确指定单个文件名时读取该文件。** 不得默认扫描或读取整个目录。

用法示例：
```
请基于 user-input/微服务升级经历.md 写一篇文章
素材：user-input/热度起来了.md
```

未指定文件时，AI 应询问用户要使用哪个文件，而不是自行选择。

## 推荐文件结构

```markdown
# 文章主题

## 我提供的素材

真实经历、数据、想法、笔记...

## 需要你额外帮我做的事情

1. 查询某些数据
2. 补充背景知识
3. 其他需求
```

## 示例文件名

- `微服务架构升级经历.md`
- `投资复盘-2026Q1.md`
- `跑步恢复心得.txt`
