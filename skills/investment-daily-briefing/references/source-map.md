# 投资观察信息源地图

> 供 `investment-daily-briefing` 使用。优先使用公开、可追溯、带明确日期的一手来源。

## 一手来源

| 类别 | 示例 |
|------|------|
| 公司公告 / IR | 公司官网 IR、港交所披露、SEC 披露 |
| 监管 / 政策 | 国务院、证监会、央行、交易所 |
| 宏观数据 | 国家统计局、美国劳工统计局、商务部 |
| 国际会议 / 央行 | 美联储、欧洲央行、G20、OPEC |

## 可辅助引用来源

| 类别 | 示例 |
|------|------|
| 权威媒体 | Reuters、Bloomberg、Financial Times |
| 专家公开言论 | 股东大会、业绩会、央行官员公开讲话 |

## 禁止直接作为公开稿依据的来源

- 匿名爆料
- 单一社交媒体传闻
- 无明确时间戳的二手转载
- 带明显买卖建议或荐股语义的观点文章
