5 月 27 日的 AI 动态围绕「变现」与「治理」两条主线展开。Meta 首次对 AI 消费者收费，Anthropic 把 Agent 安全交还企业，OpenAI 则在中期选举前夕发布安全蓝图。

## 速览

- Meta 推出 AI 订阅 Meta One Plus（7.99 美元/月）和 Premium（19.99 美元/月），下月在新加坡、危地马拉和玻利维亚启动测试
- Anthropic 发布自托管 Claude 沙箱和代码安全审计插件，企业可将 Agent 运行在自有基础设施上
- OpenAI 发布 2026 中期选举安全计划五大支柱，与美联社合作提供准确选举信息

## 重点动态

### Meta 首次对消费者收取 AI 费用

据 CNBC 和 Bloomberg 等多家媒体周三报道，Meta 宣布推出两个 AI 订阅档位。

Meta One Plus 定价 7.99 美元/月，Premium 定价 19.99 美元/月。高级版提供额外算力，可处理更大、更复杂的请求。免费版继续保留。产品负责人 Naomi Gleit 在 Instagram 视频中公布了方案细节。

CEO 扎克伯格去年 5 月曾暗示用户可以付费使用更多算力。一年后，这个方向正式落地。测试下月从新加坡、危地马拉和玻利维亚三国同时启动。

这是 Meta 首次直接对 AI 消费者收费。此前公司完全依赖广告模式，而 OpenAI、Anthropic 和 Google Gemini 早已建立付费墙。消息公布后 Meta 股价上涨约 4%。

### Anthropic 将 Agent 安全交还企业

Anthropic 在伦敦 Code w/ Claude 活动上发布了自托管沙箱（公开测试版）和 Claude Code 安全审计插件。

自托管沙箱允许 Claude Managed Agent 运行在企业自己的基础设施上，支持 Cloudflare、Daytona、Modal、Vercel 等提供商。工具执行留在企业内部——网络策略、审计日志、安全工具直接覆盖 Agent——模型编排由 Anthropic 云端负责。

安全审计插件在编辑文件、生成代码和提交时自动扫描漏洞，分析风险代码模式并审查完整 diff。Anthropic 内部数据显示，PR 安全相关评论因此减少了 30-40%。

这意味着 AI Agent 正在从 IT 看不到的盲区，变为可审计、可治理的企业应用。

### OpenAI 发布选举安全蓝图

OpenAI 周三在官方博客发布 2026 中期选举安全计划，涵盖五大支柱：传播可靠投票信息、强化选举系统网络安全、深度伪造水印检测、禁止平台被用于选举干扰，以及消除模型政治偏见。

公司已将 Codex Security 框架和 Trusted Access for Cyber 提供给各州选举官员，并向全美州务卿协会做了技术简报。

一项新增内容是与美联社合作，在 ChatGPT 中提供准确选举信息。选举安全专家 David Becker 公开认可了该计划。不过 CyberScoop 指出，SynthID 水印等部分举措此前已单独发布，OpenAI 此次是将各项安全措施整合为统一框架。

## 为什么值得关注

Meta 的 7.99 美元定价为社交 AI 建立了收费基准。选择小市场测试而非大规模推出，说明还在谨慎试探价格弹性。一旦跑通，社交网络与 AI 会员的变现模型将在全球最具分发能力的平台上铺开。

Anthropic 的自托管沙箱回答了 CIO 必须追问的问题：敏感数据到底去了哪里。让 Agent 在客户自有基础设施中执行工具调用，是企业 AI 从实验走向核心业务的最后一公里。

OpenAI 的选择安全计划在全美中期选举倒计时不足 6 个月的时间点发布。即将 IPO 的公司需要向监管和公众证明：营利与公共利益不是互斥的选项。

5 月 27 日的三条主线共同揭示：AI 行业正在从技术突破叙事，转向商业模式和治理框架的建设期。
