# 投资理财

## 类别描述

关于投资理财、金融知识、财富管理、防骗指南、财务规划等主题。

## 适用场景

- **防骗教育**：揭示常见骗局、提醒防范风险
- **理财知识**：基础理财观念、投资方法分享
- **案例警示**：真实案例解析、经验教训总结
- **理性投资**：价值投资理念、长期主义思维

## 推荐写作方向

1. **案例驱动**：通过真实故事或案例引入主题
2. **故事化表达**：用讲故事的方式传达理财理念
3. **警示性强**：强调风险，提醒保持理性
4. **通俗易懂**：避免专业术语，用简单语言讲清楚

## 映射到 Categories

- **观点评论**：对投资现象、理财观念的评论
- **学习笔记**：理财知识学习总结
- **成长故事**：理财经历、投资心路、观念转变

## 写作风格特征

- 以故事为主线，循序渐进展开
- 语言通俗易懂，贴近日常生活
- 强调警示和教育意义
- 引用真实案例增强说服力
- 情感共鸣，引发读者思考
