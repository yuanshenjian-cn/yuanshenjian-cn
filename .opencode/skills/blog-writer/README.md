# 博客写作助手 Skill

智能博客写作助手，支持交互式模式选择、风格模仿和素材融合。

## 功能特性

- **交互式创作**：主题类别、文章类型、写作风格、目标字数
- **素材输入**：支持直接粘贴或从文件读取素材
- **风格学习**：自动分析示例文章，模仿写作风格
- **智能标题**：根据内容自动提炼吸引人的标题
- **自动归档**：文章完成后保存到 `archives/` 目录

## 快速开始

```bash
# 基础用法
/blog-writer 如何使用Spring Boot构建RESTful API

# 指定类型和风格
/blog-writer Docker容器化实践 --type=技术分享 --style=轻松幽默

# 指定字数
/blog-writer Git常用命令 --type=工具推荐 --words=500
```

## 参数说明

| 参数 | 说明 | 可选值 | 默认值 |
|------|------|--------|--------|
| `--type` | 文章类型 | 技术分享、产品分析、观点评论、学习笔记、工具推荐、经验分享、成长故事 | 技术分享 |
| `--style` | 写作风格 | 专业严谨、轻松幽默、简洁明了、故事叙述、朴实稳重 | 专业严谨 |
| `--words` | 目标字数 | 500/1000/1500/2000/3000/5000 | 2000 |

## 交互式流程

调用 `/blog-writer` 后，系统会引导完成：

1. **选择主题类别**：运动健康、投资理财、生活杂谈、软件开发
2. **选择文章类型**：7种类型可选
3. **选择写作风格**：5种风格可选
4. **选择目标字数**：6档字数可选
5. **提供素材**（可选）：直接粘贴或从 `user-input/` 目录读取
6. **确认大纲**：查看并确认文章结构
7. **生成文章**：自动保存到 `archives/`

## 素材输入

**方式1：直接粘贴**
```
系统：是否要提供素材？
用户：是
用户：粘贴素材内容...
用户：素材提供完毕
```

**方式2：从文件读取**
```
系统：是否要提供素材？
用户：请从 user-input 目录读取 我的经历.md
```

## 写作风格

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| 专业严谨 | 学术化、数据支撑、逻辑严密 | 技术深度文章 |
| 轻松幽默 | 生动有趣、丰富比喻、语言活泼 | 入门教程、经验分享 |
| 简洁明了 | 短小精悍、直击要点 | 快节奏阅读 |
| 故事叙述 | 故事主线、循序渐进 | 经验分享、成长故事 |
| 朴实稳重 | 语言朴实、情绪稳定、实事求是 | 技术总结、踩坑记录 |

## 文件结构

```
blog-writer/
├── SKILL.md          # 技能定义
├── README.md         # 本文件
├── topics/           # 主题类别定义
├── categories/       # 文章类型模板
├── styles/           # 风格参考
├── samples/          # 示例文章
├── user-input/       # 用户素材目录
└── archives/         # 生成文章存档
```
