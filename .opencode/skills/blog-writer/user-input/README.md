# User Input 目录

这个目录用于存放用户输入的素材文件。

## 用途

当你有现成的素材文件时，可以直接放在这个目录，然后在写作时引用。

## 支持的文件类型

- `.md` - Markdown 格式
- `.txt` - 纯文本格式
- 其他文本格式

## 文件命名建议

使用有意义的文件名：
- `微服务架构升级经历.md`
- `产品经理的思考笔记.md`
- `Vue3学习总结.txt`

## 如何使用

在调用 blog-writer skill 时，说明要从 user-input 目录读取素材：

```
请从 user-input 目录读取 "微服务架构升级经历.md" 作为素材
```

或简写为：

```
素材：user-input/微服务架构升级经历.md
```

## 注意事项

- 素材文件应该包含你的真实经历、感想、笔记等
- 文件内容可以很随意，不要求格式完美
- AI 会自动分析和组织内容
