# 投资理财 | 热度起来了

### 我最近几天经历和感想碎片
最近 A 股也慢慢热度起来了，成交量维持在在 2 ~ 3 万亿，甚至前段时间创了新高，交易额在某天（待查询）创下了多少（待查询）。

有几个亲戚和朋友也跑来问我，让我给推荐几只基金。当我得知他们平时不关注股市，而最近经常刷到各种博主的视频和文章来教人炒股，情绪就被带动起来了。

说实话，我不太愿意直接推荐标的，一来我自己也是一个普通的业余投资者，这些标的能不能挣钱自己都没有底， 更多也只是在控制好足够的安全边际在做定投。

而且我个人也是那种长期的投资风格，推荐的标的很可能几年都不挣钱需要主动买套的那种， 但他们可能连投资最基础的认知还没有建立，主要是因为最近看到身边的人情绪高涨，担心错够了这一波财富，就入场捞一把。

根据市场规律，当股市涨起来的时候，就会有各种牛鬼蛇神直播讲课写文章来教人炒股，这些平时不关注股市的人就比较容易被他们煽动。 一开始他们跟着小投一些，然后发现还真的涨了，就会自信膨胀，接着就加大投入，结果一旦市场调整就会慌张割肉。

这种在牛市里亏钱的故事这些年我也听得太多了，大都是在热度起来，漫天都是赚钱的情绪的时候，他们一开始进来从小钱试水逐步加大投入，一天可能能够挣到几个月工资，此时人的心态，结果大部分难免亏钱割肉，要闷套牢盼着回本。

我没有直接推荐，其实我更希望推荐他们先花点时间学习一些投资的基础知识，了解市场的运作机制和风险管理，股市不缺机会，缺的是 恰当的认知和合适的钱。

我倾向推荐了一些常年都在践行价值逆向投资的博主，比如之前我哥给我推荐的银行螺丝钉，他算是一个布道者和践行者，他长期的投资收益率至少在15%+，普通散户跟着他定投很省心，不过也要在熊市的时候敢于跟投。比较典型的案例是我大姐之前跟着他定投一只基金，但是后来因为继续下跌她没继续投了，现在螺丝钉已经挣了不少收益了，她的才接近回本。

另外我也更愿意新手去读几本投资入门的书，建立一些基础理念，学习一些必要基础知识之后，了解投资的基本原理和市场主流的标的构成，这些才是构成让自己敢于逆风前进的认知和底气。
