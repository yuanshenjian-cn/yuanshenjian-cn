# 写作禁止清单

> 每次写作前必须读取本文件，并在写作全程严格遵守。

---

## 格式禁止

- **子标题不能有序号**：禁止 `## 一、` `## 二、` `### 2.1` `### 1.1` 等一切序号前缀
- **标题与子标题不能使用破折号**：禁止 `## 标题——副标题` `### 概念——定义` 等一切在标题中使用 `——` 或 `-` 连接的写法
- **禁止章节间分割线**：不能在 `##` 级章节之间插入 `---` 分割线；文章层次由标题本身承担，分割线会让页面产生割裂感，破坏阅读节奏
- **避免堆砌长句**：段落控制在 80 字以内为宜，超过时优先考虑换行或拆分；强调语气的长句可酌情保留，但禁止用逗号串联三个以上从句
- **禁止列表替代分析**：不能用"首先…其次…再次…最后"结构替代真正的论证段落

---

## 词语禁止

以下词语/短语禁止出现：

| 禁止 | 可替代为 |
|------|---------|
| 总的来说 | 直接写结论 |
| 综上所述 / 综上 | 直接写结论 |
| 值得一提的是 | 直接陈述 |
| 值得注意的是 | 直接陈述 |
| 深入探讨 / 全面分析 / 深度解析 | 具体说做了什么 |
| 在当今 XX 背景下 | 直接进入主题 |
| 至关重要 / 尤为重要 / 不可或缺 | 说清楚为什么重要 |
| 不难发现 / 由此可见 | 直接陈述结论 |
| 可以说 / 可谓 | 直接陈述 |
| 毋庸置疑 | 直接陈述 |

---

## 开头/结尾禁止

- **禁止套路式开头**：不能用"首先介绍背景，然后分析问题，最后给出建议"结构
- **禁止客套结尾**：不能出现"希望对你有所帮助"、"欢迎留言讨论"、"感谢阅读"等
- **禁止空洞总结**：结尾章节不能只是重复前面说过的内容
